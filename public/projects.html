My Plans

Food Blog Website

I am working on a group project for university. The original project was a UX project to design a high-fidelity prototype however a few of us decided to start develop that website into a fully working website, complete with an API with authentication with JWT and a MongoDB database.

This Website - bnewman.dev

This website is currently written purely in HTML, CSS and JS with future plans to re-write in vue.js. The domain points to a VPS server running nginx listening on port 443 with an SSL certificate issued by letsencrypt. Nginx is set up as a reverse proxy server which directs incoming traffc to a Node.js server which (at the moment) statically serves the pages.
